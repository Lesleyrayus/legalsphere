# **App Name**: Law Hub

## Core Features:

- Task Management: Accept or deny tasks assigned by the admin.
- User Chat: Real-time chat functionality with users.
- Booking Display: Display a list of user bookings with details.
- Notification System: Display notifications related to tasks, bookings, and user messages.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Teal (#009688) for interactive elements and highlights, creating a sense of efficiency and clarity. It's saturation makes it highly distinct from the primary and background.
- Body and headline font: 'Inter' sans-serif for clear and professional readability.
- Use professional and consistent icons for navigation and task representation.
- Dashboard layout inspired by the provided admin dashboard, focusing on clear data presentation and easy navigation.
- Subtle transitions and animations to enhance user experience, providing feedback on actions.
